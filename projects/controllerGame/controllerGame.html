controller game

This is a little game I wrote for the control engineers among you. The JavaScript function "controller()" returns a force which is applied to the cart. The goal of the game is to use the "controller()" function to swing up the pendulum and stabilize it in the upper position. Press "execute" to test your solution. Press "reset" to bring the pendulum back to the starting position. Feel free to share your solution via email. I'm locking forward to see your creative ideas. In case you need some inspiration, you can find my solution here. If you are wondering why my solution works, here is a nice paper which explains the energy based swing up.
