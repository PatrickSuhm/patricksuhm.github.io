Regelungstechnik Spiel

Hier habe ich ein kleines Regelungstechnik Spiel entworfen. Das Ziel des Spiels ist es, das Pendel mithilfe der JavaScript-Funktion "controller()" aufzuschwingen und es dann in der oberen Position zu stabilisieren. Die "controller()"-Funktion gibt eine Kraft zurück, die auf den Schlitten wirkt. Der Button "execute" führt den Code aus. Mit "reset" wird das Pendel in eine zufällige Ausgangsposition zurück gebracht. Falls jemand Inspiration benötigt, meine Lösung gibt es hier.
